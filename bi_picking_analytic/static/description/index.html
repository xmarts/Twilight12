Analytic Account on Stock Picking with Analytic Tags

This module is useful to create analytic account at time of delivery. Also you can add analytic account in Purchase at time of requesting a Quotation for new products in Odoo.

How to use

The anlytic account you defined in sales order will pass in delivery order with the anlytic tags. and when you done the delivery order it will create account analytic entry.





The sale price of the product will be recorded in account analytical line.


You can add multiple analytical accounts for all purchase quotations.


Analytical Account details are displayed in picking of particular purchase order for each products available.
